Void Bot – Privacy Policy

Last Updated: July 29, 2025

This Privacy Policy explains what data Void ("the Bot", "we", "our", or "us") collects, how we use it, and how we keep it safe. By using Void, you agree to the practices described below.

Information We Collect

When you use Void, we may collect and process certain Discord‑provided data in order to perform tag searches, server tag tracking, and usage-based enhancements. This may include:

User Data

  • Discord usernames, IDs, and tags that are searched or tracked.
  • Commands you issue to the Bot (including message content for command parsing and contextual replies).
  • Presence/activity status of users (used for tracking tag usage in profiles).

Server Data

  • Server IDs where Void is added.
  • Server tags, metadata, and tag associations used for tracking.
  • Server-specific vanity representations appearing in user profiles.

Log Data (for debugging and moderation)

  • Timestamps of commands and interactions.
  • Internal error or performance logs.

Why We Track Message Content

Void uses limited message content tracking solely to:

  • Parse slash and context commands (when applicable).
  • Generate dynamic help embeds or contextual Bot responses.
  • Ensure proper functionality of features like /taglist, /tagsearch, or when handling mentions of tracked tags within user input.

We do not read, store, or analyze private conversations or unrelated messages. Only messages directly interacting with the Bot are processed, and only the minimum necessary content is read to perform the function.

Why We Track Presence Information

Void uses presence and activity tracking to:

  • Identify when a user is displaying server-specific tags, vanity URLs, or emojis in their profile.
  • Track server representation in user statuses (e.g., when a user promotes your community).
  • Support moderation or analytics features based on server presence trends.

This allows Void to give accurate insight into how many users are actively promoting or representing your server at a given time.

Why We Track Primary Guild Changes

Void monitors primary guild changes from member update events in order to:

  • Detect when a user removes or adds a server tag in their status.
  • Automatically adjust tracking for server tag engagement (such as tracking how many users have the server tag "equipped").

This supports accurate tag usage metrics and community tracking across multiple servers.

How We Use the Data

We only use collected data to:

  • Provide and improve the Bot's tag search, analytics, and tracking features.
  • Detect when server vanity tags are being displayed in user statuses.
  • Maintain stability, monitor abuse, and fix bugs.
  • Comply with Discord's Developer Terms of Service and applicable laws.

We Do Not:

  • Sell or share your data with third parties for marketing.
  • Collect passwords, private messages, or any non‑Discord data.
  • Integrate with non-Discord platforms or third-party advertising services.

Data Storage & Retention

Data is stored only as long as necessary for Void to function properly and for maintenance or debugging purposes.

We may occasionally retain anonymized or aggregated usage data (e.g., command usage frequency or tag popularity) for improving features.

If you wish to have certain data (such as search logs or tracked tags) removed, please contact us via our support server or email.

Data Sharing

We do not share your data unless:

  • Required by law or legal process.
  • Necessary to comply with Discord's policies.
  • Needed to investigate abuse, security incidents, or spam.

Security

We take reasonable steps to protect all collected data:

  • Secure, limited-access databases.
  • Regular auditing for unauthorized access.
  • Bug and error monitoring to patch vulnerabilities.

However, no system is 100% secure. We cannot guarantee absolute protection, but we strive to follow best security practices.

Your Choices & Rights

  • Opt‑Out: You may stop using the Bot or remove it from your server at any time.
  • Data Requests: You may request to review or delete any personal or server-related data associated with your usage of Void.

Third‑Party Services

Void operates through the Discord platform and adheres to Discord's Privacy Policy. We do not integrate with unrelated third-party services.

Changes to This Policy

We may revise this Privacy Policy periodically. Significant changes will be posted in our support server or announced via Bot message. Continued use of the Bot after updates means you accept the revised terms.
